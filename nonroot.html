If you are having trouble downloading files from vanced.app, try:

1) 1-Time Solution
• Download the file in Incognito Mode.

2) Permanent Solution
• Go to System Settings > Apps > Google Drive > Open by default and set Open supported URLs to Always ask.
Try downloading the file again and when prompted, choose to download the file using your browser, not Google Drive.
